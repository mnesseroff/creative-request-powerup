Creative Request Power-Up for Trello

Status: Power-Up is ready to be installed in Trello

Features

  • Custom request form for creative projects
  • Automatic card creation in your Trello board
  • Video format specifications
  • Asset link collection
  • Due date tracking
  • Reference materials management

How to Use

  1. Add this Power-Up to your Trello board
  2. Click the "Create Request" button on any card
  3. Fill out the creative request form
  4. Submit to automatically create a new card with all details

This Power-Up helps streamline creative requests and project management in Trello.
